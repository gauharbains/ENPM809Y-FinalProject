Creates a texture from a bitmap.

The texture will be loaded upside down because tdogl::Bitmap pixel data is ordered from the top row down, but OpenGL expects the data to be from the bottom row up.

Parameters
bitmapThe bitmap to load the texture from
minMagFilerGL_NEAREST or GL_LINEAR
wrapModeGL_REPEAT, GL_MIRRORED_REPEAT, GL_CLAMP_TO_EDGE, or GL_CLAMP_TO_BORDER
