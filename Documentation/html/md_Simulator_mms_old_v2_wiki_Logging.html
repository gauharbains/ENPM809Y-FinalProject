Logging

These functions print to stdout and log to a file in the run/ directory. The levels are somewhat arbitrary. The convention that I use is that errors should cause a failure, while warnings should allow the program to continue.

1 {c++}
2 void debug(const std::string& str);
3 void info(const std::string& str);
4 void warn(const std::string& str);
5 void error(const std::string& str);

Example:

1 {c++}
2 // You can log a std::string
3 std::string foo("Hello World!");
4 mouse->warn(foo);
5 
6 // Or a string literal
7 mouse->info("Moving forward");
8 
9 // Use std::to_string() for numeric values
10 mouse->debug(std::to_string(4));
